1) install the latest version of vim
	* sudo add-apt-repository ppa:jonathonf/vim
2) inside the folder .vim create a folder named "bundle"
3) download the plugin YouCompleteMe: https://github.com/ycm-core/YouCompleteMe
	* git submodule update --init --recursive
	* python3 install.py --clang-completer
4) create system tags inside the folder .vim:
	* ctags -R --c++-kinds=+p --fields=+iaS --extra=+q /usr/include /usr/local/include/ (add other folders if needed)
5) put the file .ycm_extra_conf.py inside the folder .vim
6) create a folder autoload inside the .vim folder and then:
	* curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
7) launch vim . and execute :PlugInstall
8) replace in the folder .vim/plugged/vim-snippets/UltiSnips/ the file cpp.snippets
9) the first time you open a project with vim press Ctrl+l for creating new tags and call cmake with: cmake -DCMAKE_EXPORT_COMPILE_COMMANDS=ON ..
